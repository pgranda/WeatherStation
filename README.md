# WeatherStation

## Project:
Weather Station

## Author:
Pawel Granda

## Date Created:

## Brief:
Weather station made using Arduino board, LCD Display, Rain Sensor and DHT11 Humidity and Temperature Sensor.
Libraries for DHT11 and LCD Display can be downloaded from their developers.
